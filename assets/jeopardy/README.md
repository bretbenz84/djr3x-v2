Jeopardy clue data
==================

`clues.tsv` is a local playable clue bank for DJ-R3X's verbal Jeopardy-style
game. It was pulled from jwolle1's `jeopardy_clue_dataset` repository:

https://github.com/jwolle1/jeopardy_clue_dataset

This copy uses the Kids/Teen tournament subset rather than the full combined
Season 1-41 file so the application has thousands of real clues without adding
the full 74 MB combined dataset to the repo.

The upstream project notes that the Jeopardy clue data is property of Jeopardy
Productions, Inc. and should not be used for a public-facing product. This local
copy is intended for private, personal use inside DJ-R3X.
